Django Ecommerce - How Start Project 

1. Activate the virtual environment:
Type =  source venv/bin/activate

2. Start the Django development server:
Type =  python manage.py runserver

After the server starts, open or click:
http://127.0.0.1:8000/

When you're finished or closing the project:
Type =  Ctrl + C
